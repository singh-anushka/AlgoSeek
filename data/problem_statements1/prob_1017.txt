
Given a positive integer n, write a program to print the last digit of n.
For example, if n=1234, then print 4.
Input:
A positive integer n.
Output:
Output the last digit of n.
Constraints

1≤n≤105

Sample Input:
1234
Sample Output:
4
EXPLANATION:
Given n=1234, the output will be last digit i.e. 4.
